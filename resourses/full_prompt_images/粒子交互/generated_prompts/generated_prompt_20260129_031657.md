# 3D 粒子幻境交互系统开发提示词

你是一位精通 WebGL、Three.js 和创意交互的前端开发专家。请根据以下分析和要求，编写一个单文件（Single-file）的 HTML 解决方案，实现视频/截图中的粒子交互效果。

### 1. 视觉与交互分析
根据提供的图片，项目呈现出以下核心特征：
*   **核心视觉**：一个由成千上万个发光粒子组成的球体或星系结构。中心呈现高密度的金黄色/橙色核心，外围环绕着白银色/冷色调的稀疏粒子环。
*   **光影质感**：粒子具有强烈的自发光效果，使用了光晕（Bloom）后处理技术，背景为深黑，对比度高，呈现出“幻境”般的科幻感。
*   **空间形态**：结构类似于土星环或旋涡星系，具有明显的 3D 深度感。
*   **交互方式**：图片展示了手势控制（张开手掌旋转/拉近，捏合手势缩放）。在 Web 端实现时，需将这些手势物理映射为平滑的鼠标/触摸交互（拖拽旋转、滚轮缩放），并增加惯性阻尼感以模拟手势操作的“悬浮感”。
*   **UI 界面**：
    *   左上角：半透明 HUD 面板，显示标题“粒子·幻境”及系统参数。
    *   左下角：控制按钮组（重置、自动旋转等）。
    *   右上角：全屏及设置图标。

### 2. 开发要求详情

请生成一个包含 HTML, CSS, 和 JavaScript 的完整 `index.html` 文件。

#### A. 技术栈与环境
*   **核心库**：使用 Three.js (通过 CDN 引入)。
*   **后处理**：使用 Three.js 的 EffectComposer 和 UnrealBloomPass 实现发光效果 (通过 CDN 引入)。
*   **文件结构**：所有代码（样式、逻辑、结构）必须内嵌在同一个 HTML 文件中，无外部本地依赖，确保在浏览器直接打开即可运行。

#### B. 视觉效果实现细节
1.  **粒子系统 (Particle System)**：
    *   创建两个主要的粒子群：
        *   **核心球体**：约 15,000 个粒子，分布在球体表面和内部，颜色为金黄色/橙色 (`#ffaa33`)。
        *   **外围星环**：约 5,000 个粒子，呈圆环状或扁平圆盘状分布，颜色为银白色/淡蓝色 (`#ffffff` 或 `#aaccff`)。
    *   **材质**：使用 `THREE.PointsMaterial`，开启 `vertexColors`（顶点颜色），设置 `sizeAttenuation: true`（大小随距离衰减），并使用 `AdditiveBlending`（叠加混合模式）以增强发光感。
    *   **动态纹理**：通过 Canvas 动态生成一个圆形径向渐变的贴图作为粒子的 `map`，使粒子看起来柔和而非方形。

2.  **后期处理 (Post-Processing)**：
    *   必须实现 **Bloom (辉光)** 效果。配置 `threshold` (阈值), `strength` (强度 ~1.5), `radius` (半径) 以达到截图中那种朦胧的亮光感。

3.  **动画与特效**：
    *   **自转**：整个粒子系统应有一个缓慢的自动自转。
    *   **呼吸感**：粒子的大小或透明度可以有微弱的正弦波波动，模拟“呼吸”的生命感。
    *   **入场动画**：加载时粒子从散乱状态汇聚成球体，或透明度渐显。

#### C. 交互功能 (模拟手势)
由于无法直接在该环境中集成复杂的摄像头手势识别模型，请使用**高拟真鼠标/触控交互**来复刻视频中的手势效果：
*   **旋转 (模拟手势挥动)**：鼠标左键拖拽 / 手指滑动，控制摄像机围绕中心旋转。必须启用 `enableDamping` (阻尼)，设置较大的阻尼系数，使旋转具有物理惯性，感觉像是在拨动悬浮的能量球。
*   **缩放 (模拟手势捏合)**：鼠标滚轮 / 双指捏合，控制视距远近。缩放也需要平滑过渡。
*   **鼠标跟随**：粒子群受鼠标位置微弱影响，产生轻微的视差偏移。

#### D. UI 界面 (HUD)
*   实现科幻风格的 HUD 界面（绝对定位，覆盖在 Canvas 上）。
*   **左上角**：标题“粒子·幻境 (PARTICLE ILLUSION)”，下附几行伪代码或参数显示（如“SYSTEM READY”, “PARTICLES: 20000”）。
*   **左下角**：一排半透明玻璃拟态按钮，功能包括：“重置视角”、“切换颜色”、“自动旋转开关”。
*   字体使用无衬线字体（如 Roboto, Arial），颜色为白色，带轻微文字阴影。
