你需要生成一个名为 **"日签 (Daily Mood)"** 的单页面应用。请编写一个包含所有 HTML、CSS (Tailwind) 和 JavaScript 的 `index.html` 文件。

### 1. 整体效果描述
创建一个极简、优雅、具有沉浸感的“日签”风格网页。页面核心是一张高质量的风景/花卉摄影图，经过模糊和压暗处理作为背景。页面中央悬浮着白色的日期和名言文字，排版讲究，类似杂志或日历的设计。底部左侧有音乐播放入口，右上角有刷新按钮。交互时带有平滑的淡入淡出动画。

### 2. 技术栈与库引用 (CDN)
请在 `<head>` 中通过 CDN 引入以下库：
- **Tailwind CSS 3.0+**: 用于快速构建响应式布局和样式。
- **Font Awesome 6**: 用于图标（刷新按钮、音乐图标）。
- **Anime.js**: 用于处理文字和背景切换时的平滑过渡动画。
- **Google Fonts**: 引入 `Noto Serif SC` (思源宋体) 或 `ZCOOL XiaoWei`，确保中文字体优雅古朴。
- **Lunar-javascript** (可选，若无轻量库，可使用简易算法或 mock 数据): 用于显示农历日期。

### 3. 视觉设计细节
- **背景层**:
  - `<img>` 标签全屏覆盖 (`object-cover`, `w-full`, `h-full`)。
  - 添加 `brightness-75` (25% 黑色遮罩) 和 `blur-sm` (轻微高斯模糊) 类，确保背景不抢眼且文字清晰。
  - 图片源：使用 Unsplash Source API (如 `source.unsplash.com/random/1920x1080/?nature,flower,landscape`) 或预设一组高质量图片数组进行随机切换。
- **排版**:
  - 字体颜色：纯白 (`text-white`)，带有轻微阴影 (`drop-shadow-md`) 增加对比度。
  - 字体风格：衬线体 (Serif)，营造文学感。
- **布局**:
  - 使用 Flexbox 或 Grid 绝对居中主要内容。
  - 顶部右上角：刷新按钮 (Icon)。
  - 底部左下角：音乐播放器触发器。

### 4. 功能模块与逻辑
#### A. 时间显示模块 (居中)
- **第一行**: `MM月dd日` (例如 "07月11日")，字体大小 `text-lg`，透明度稍低。
- **第二行**: `星期X · 农历XX` (例如 "星期五 · 农历六月十四")，字体更小 `text-sm`，字间距略宽。
- **核心行**: 当天日期数字 (例如 "11")，**超大字体** (`text-8xl` 或更大)，字重细或适中，作为视觉重心。

#### B. 名言展示模块 (日期下方)
- **内容**: 随机从预设数组中选取（包含励志、哲学、古诗词）。
- **样式**: 居中对齐，`text-xl` 或 `text-2xl`，行高舒适 (`leading-relaxed`)。
- **署名**: 位于名言右下方，右对齐，格式为 `—— 作者，出处`，字体较小且有透明度。

#### C. 音乐播放功能 (左下角)
- 一个简洁的圆形图标按钮。
- 点击后展开或弹出一个嵌入的 Spotify iframe (预设一个专注/白噪音 Playlist ID) 或 HTML5 Audio 播放器。
- 初始状态为收起或仅显示图标。

#### D. 交互与动画
- **加载/刷新**: 页面加载或点击刷新按钮时：
  1. 使用 Anime.js 让当前内容淡出。
  2. 更换背景图、日期（如果是新的一天）、名言。
  3. 内容淡入，日期数字可以有一个轻微的向上浮动或缩放效果。
- **响应式**: 移动端字体适当缩小，布局自适应。

### 5. 代码要求
- **单文件交付**: 所有 HTML 结构、Tailwind 类名配置、自定义 CSS (如有)、JavaScript 逻辑必须合并在一个 `index.html` 文件中。
- **数据模拟**: 在 JS 中内置一个包含至少 10 条高质量中外名言的数组 (对象包含 content 和 author)。
- **健壮性**: 确保图片加载失败时有回退颜色，确保 API 不可用时页面不崩坏。
