# Web开发提示词：鹈鹕骑自行车 SVG 动画页

请作为一名精通SVG图形设计和前端动画的高级Web开发人员，编写一个完整的单页应用。该项目需要在一个独立的 `index.html` 文件中展示一只鹈鹕（Pelican）骑自行车的动态场景。

## 1. 整体效果描述
创建一个充满趣味性和艺术感的网页，核心内容是一个居中的SVG插画。画面中，一只卡通风格的鹈鹕正在努力地骑着一辆自行车。整体视觉风格应采用扁平化设计（Flat Design）或微渐变风格，色彩明快鲜艳，营造出轻松愉快的氛围。页面应包含控制面板，允许用户与动画进行交互。

## 2. 视觉效果细节
### 鹈鹕（角色）
- **外观特征**：鹈鹕应具有标志性的大喉囊（喙部），羽毛主要为白色或浅灰色，喙部为橙黄色。
- **配饰**：为了增加趣味性，请给鹈鹕戴上一顶复古飞行员护目镜或一顶鲜艳的小帽子，脖子上围一条随风飘扬的围巾。
- **姿态**：鹈鹕的双脚（带蹼）应放置在自行车的脚踏板上，翅膀像手一样握住车把手，身体微微前倾，呈现出专注骑行的姿态。

### 自行车（道具）
- **设计**：经典复古款式的自行车，具有明显的车架结构、两个辐条轮和车把。
- **配色**：车身颜色应与鹈鹕形成对比（例如复古绿、深红或海军蓝）。

### 环境与背景
- **地面**：简单的线条或色块表示道路。
- **背景**：淡雅的天空背景，包含云朵或简单的树木剪影，用于产生视差滚动效果以体现速度感。

## 3. 动画与特效说明
请使用 CSS3 Animation 和 SVG 变换（Transforms）实现以下动画：
- **车轮转动**：前后车轮应持续旋转，辐条模糊或旋转以表现速度。
- **蹬踏动作**：这是最关键的动画。鹈鹕的腿部（大腿、小腿）和脚掌必须跟随脚踏板的圆周运动进行机械联动（模拟反向运动学 IK 效果）。脚踏板和曲柄需要围绕中轴旋转。
- **身体起伏**：鹈鹕的身体和头部应随着骑行节奏有轻微的上下起伏（Bobbing），增加生动感。
- **围巾飘动**：围巾尾部应有波浪状的摆动动画，模拟风吹的效果。
- **背景运动**：地面的纹理或路边的景物向后移动，形成无限循环的滚动效果，增强骑行的视觉错觉。

## 4. 交互功能说明
在页面底部或角落提供一个简洁的控制面板：
- **速度滑块（Range Slider）**：允许用户调节骑行速度。滑动滑块应实时改变所有相关动画（车轮、蹬踏、背景移动）的播放速度（`animation-duration`）。
  - 最左侧：停止或慢动作。
  - 最右侧：极速冲刺模式。
- **主题切换**：一个按钮用于切换“白天/黑夜”模式，改变背景天空颜色和SVG的整体亮度。

## 5. 技术实现要点
- **文件结构**：**必须**将所有的 HTML 结构、CSS 样式和 JavaScript 逻辑合并在一个名为 `index.html` 的文件中。
- **SVG 构建**：
  - 使用内联 SVG (`<svg>`) 代码，不要引用外部图片。
  - 代码需整洁，对 SVG 元素（如 `<g id="wheel">`, `<g id="leg-left">`）进行合理的 ID 命名和分组，以便于 CSS 选择器定位。
- **响应式设计**：SVG 应具有良好的缩放性，确保在桌面端和移动端浏览器中均能完整、居中显示。
- **动画性能**：尽量使用 `transform` 和 `opacity` 属性进行动画，以确保 60fps 的流畅度。
- **JavaScript 逻辑**：
  - 监听滑块输入事件，通过修改 CSS 自定义属性（CSS Variables，例如 `--cycle-speed`）来动态控制动画时长。

请生成包含上述所有功能的完整代码。
