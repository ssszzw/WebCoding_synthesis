# 提示词：构建 HTML5 复古霓虹风格太空射击游戏 "Grid Destroyer"

请编写一个完整的、包含在一个 `index.html` 文件中的 HTML5 Canvas 射击游戏。该游戏基于经典的复古合成波（Synthwave/Retrowave）美学，复刻名为 "Grid Destroyer" 的视觉风格和玩法。

### 1. 整体视觉风格与界面设计
*   **色调主题**：采用高对比度的霓虹配色方案。
    *   主色调：霓虹青色（Cyan, #00FFFF）和霓虹品红（Magenta, #FF00FF）。
    *   背景色：深青色或暗蓝绿色渐变，营造深邃太空感。
*   **边框设计**：游戏画布周围需要有一个粗的、发光的品红色霓虹边框。
*   **背景效果**：
    *   **伪 3D 透视网格**：背景必须是一个具有透视感的流动网格（Perspective Grid）。纵向线条汇聚于屏幕中心稍上方的一个消失点，横向线条不断从上向下移动，模拟飞船高速向前飞行的视觉错觉。
    *   网格颜色为暗青色，带有微弱的发光效果。
*   **UI 界面（HUD）**：
    *   **顶部栏**：使用青色线条框定。
    *   **左侧**：显示 "SCORE: [数值]"，下方有品红色的能量/生命条。
    *   **中间**：显示游戏标题 "GRID DESTROYER"（红色或品红色字体，机械科幻风格）。
    *   **右侧**：显示 Boss/敌人的状态图标（一个发光的五边形，带有黄色圆点装饰）。
    *   **底部**：显示操作指引文本 "ARROW KEYS / WASD: MOVE | SPACE: SHOOT | SHIFT: BOOST"，字体为复古单色等宽字体，颜色为青色。

### 2. 游戏实体与绘制细节
所有图形元素必须通过 Canvas API 原生绘制，**不使用外部图片素材**。

*   **玩家飞船**：
    *   形状：一个简单的发光线框三角形，类似纸飞机形状。
    *   颜色：青色线条，中心可能有白色高光。
    *   尾焰：移动时飞船底部有简单的粒子喷射效果。
*   **子弹**：
    *   玩家子弹：细长的青色激光束，高射速。
    *   敌人子弹：品红色或紫色的发光球体/短光束。
*   **敌人 (Grid Destroyer Boss)**：
    *   形状：一个巨大的五边形结构，类似图片右上角的图标。
    *   细节：中心有发光核心，周围有环绕的防御部件。
    *   颜色：以红色/品红为主，带有黄色高光。

### 3. 交互与游戏机制
*   **控制方式**：
    *   `WASD` 或 `方向键`：控制飞船在屏幕范围内上下左右移动。
    *   `空格键 (SPACE)`：发射激光（按住可连发）。
    *   `Shift 键`：加速移动（Boost 模式），此时飞船尾焰变大。
    *   鼠标支持（可选）：飞船跟随鼠标位置移动，左键射击。
*   **战斗逻辑**：
    *   **Boss 行为**：Boss 悬浮在屏幕上方，左右移动，并定期发射散射弹幕或瞄准玩家的子弹。
    *   **碰撞检测**：玩家子弹击中 Boss 扣血，Boss 变色闪烁；玩家被击中扣除生命条。
    *   **分数系统**：每次击中敌人增加分数。
*   **特效与动画**：
    *   **发光效果**：使用 `shadowBlur` 和 `shadowColor` 实现所有元素的霓虹辉光感。
    *   **粒子爆炸**：当子弹击中敌人或玩家被击中时，产生碎片粒子爆炸效果。
    *   **屏幕震动**：玩家受到伤害或 Boss 被击毁时，画面轻微震动。

### 4. 技术实现要求
*   **文件结构**：**必须**将 HTML 结构、CSS 样式和 JavaScript 逻辑全部合并在一个 `index.html` 文件中。
*   **Canvas**：使用 HTML5 Canvas 渲染所有游戏画面。确保 `requestAnimationFrame` 用于平滑的 60fps 动画循环。
*   **响应式**：游戏区域应在浏览器窗口中居中，并保持固定的宽高比（例如 4:3），或者自适应窗口大小但保持内容不拉伸。
*   **字体**：使用 CSS 引入类似于 "Courier New", "Orbitron" (Google Fonts) 或系统等宽字体，以此强化黑客/终端风格。

### 5. 代码生成结构参考
请按以下结构编写代码：
1.  **CSS**：定义页面背景（黑色）、Canvas 居中、霓虹边框样式、字体样式。
2.  **HTML**：Canvas 标签及容器。
3.  **JS 类/对象**：
    *   `Game`：主控制器，处理循环、分数、状态。
    *   `InputHandler`：监听键盘事件。
    *   `Player`：处理玩家位置、绘制、边界限制。
    *   `Projectile`：处理子弹运动。
    *   `Enemy`：处理 Boss 逻辑和绘制。
    *   `Particle`：处理爆炸特效。
    *   `Background`：专门处理透视网格的数学计算和绘制。

**请生成完整的、可直接运行的 index.html 代码。**
