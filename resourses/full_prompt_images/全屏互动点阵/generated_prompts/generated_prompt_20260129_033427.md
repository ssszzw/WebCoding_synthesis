请创建一个基于 HTML5、TailwindCSS 和 P5.js 的单文件（index.html）全屏互动点阵动画生成器。该文件需包含所有必要的 CSS 样式、HTML 结构和 JavaScript 逻辑，确保在现代浏览器中直接打开即可完美运行。

以下是详细的需求说明和技术规范：

### 1. 整体架构与技术栈
- **核心库**：通过 CDN 引入 P5.js (最新版) 和 TailwindCSS (v3.4+ script)。
- **布局**：
  - **背景**：全屏 P5.js 画布 (`z-index: 0`)，固定定位。
  - **UI 层**：右侧浮动控制面板 (`z-index: 10`)，采用玻璃拟态设计（半透明背景 + 背景模糊）。
- **主题支持**：实现完整的深色模式（Dark Mode）和浅色模式切换。默认检测 `prefers-color-scheme`，并提供手动切换按钮。
  - 浅色模式：背景近白，点阵主色为蓝色 (`#3B82F6`)，面板背景白色半透明。
  - 深色模式：背景深灰 (`#111827`)，点阵主色为浅蓝或青色，面板背景深灰半透明。

### 2. P5.js 核心逻辑与动画实现
- **网格系统**：
  - 创建一个虚拟的巨大网格系统，其总覆盖面积至少是当前视口（Viewport）面积的 10 倍，以确保无论屏幕如何旋转或缩放，网格都足够大。
  - **性能优化（关键）**：在 `draw()` 循环中，**仅渲染**当前视口可见区域内（加上少量边缘缓冲）的点。不要渲染屏幕外的点，以保证高帧率。
- **点阵属性**：
  - 每个点需根据所选形状绘制（圆形、正方形、十字、三角形、菱形/星形）。
  - 点的大小和颜色透明度应根据动画模式动态计算。
- **动画模式算法**：
  1.  **波浪 (Wave)**：基于正弦波 (`Math.sin`)，结合 x/y 坐标和时间因子 (`frameCount`)，产生对角线方向的起伏效果。
  2.  **脉冲 (Pulse)**：基于中心点向外扩散的周期性缩放效果，类似呼吸灯。
  3.  **涟漪 (Ripple)**：以屏幕中心或鼠标位置为原点，产生向外扩散的波纹，距离中心越远衰减越大。
  4.  **噪声 (Noise)**：使用 P5.js 的 `noise()` 函数，生成有机的、云雾状的流动效果。

### 3. 用户界面 (UI) 设计与交互
控制面板应位于屏幕右侧，宽度约 320px，包含以下模块（从上到下）：

- **头部**：标题“控制面板”与深色/浅色模式切换按钮（使用月亮/太阳图标）。
- **动画模式选择**：
  - 2x2 网格布局的按钮组：[波浪] [脉冲] [涟漪] [噪声]。
  - 选中状态需高亮显示（如蓝色背景，白色文字）。
- **形状选择**：
  - 2x3 或 3x2 网格布局的图标按钮：[圆形] [方形] [十字] [三角形] [菱形] [星形]。
  - 按钮内部使用 CSS 绘制或 SVG 图标展示对应形状。
- **参数控制（滑块组）**：
  - 每个参数包含：标签、滑块 input (`type="range"`)、当前数值显示。
  - **点阵密度 (Density)**：控制网格行/列的总数（范围 10-100）。
  - **形状大小 (Size)**：控制基准点的大小（范围 1-20）。
  - **动画速度 (Speed)**：控制时间增量的倍率（范围 0.1-5.0）。
  - **网格间距 (Spacing)**：控制点与点之间的像素距离（范围 10-100）。
- **底部信息栏**：
  - 显示实时统计数据：
    - "矩阵覆盖: 10.0x 视口"
    - "总点数: [计算值]"
    - "可见点数: [实时渲染数量]"

### 4. 响应式设计要求
- **桌面端**：控制面板固定在右侧，有合适的内边距和阴影。
- **移动端**：
  - 控制面板默认折叠到底部或缩成一个悬浮按钮（FAB）。
  - 点击后弹出/展开面板。
  - 字体和触摸目标（按钮、滑块）需适配手指操作。
  - 确保画布在移动端全屏显示，禁止页面默认的滚动行为（`overflow: hidden`）。

### 5. 代码实现细节要求
- 将 CSS 放在 `<style>` 标签中（除了 Tailwind 类名外，自定义滚动条或特定动画样式）。
- 将 JS 逻辑放在 `<script>` 标签中。
- 使用 P5.js 的 `windowResized()` 函数处理窗口大小改变，确保画布始终填满屏幕。
- 初始化时，默认选中“波浪”模式和“圆形”形状。
- 代码结构清晰，变量命名规范，包含必要的注释。
- 确保所有交互（滑块拖动、模式切换）立即反映在画布动画上，无延迟。

请根据以上所有要求，生成完整的、可运行的 HTML 代码。
