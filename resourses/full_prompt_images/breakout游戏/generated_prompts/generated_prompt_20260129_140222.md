# Breakout Game Generation Prompt

请编写一个包含 HTML、CSS 和 JavaScript 的单文件 `index.html`，实现一个基于 HTML5 Canvas 的打砖块（Breakout）游戏，完美复刻截图中的视觉风格和玩法。

### 1. 整体描述
创建一个复古街机风格的打砖块游戏。背景深色，砖块色彩鲜艳，具有霓虹质感。游戏核心包含球体物理运动、鼠标控制挡板、砖块消除及分数统计。

### 2. 视觉效果细节
*   **背景**：深蓝色或黑色纯色背景。
*   **砖块**：
    *   位于屏幕上方，排列整齐。
    *   约 7-8 行，每行颜色不同（从上到下：红、橙、黄、绿、青、蓝、紫）。
    *   砖块需有简单的光泽感或渐变填充，模拟微立体效果。
*   **挡板**：底部绿青色矩形条，扁平化设计。
*   **球体与特效**：
    *   白色小球。
    *   **关键特性**：球体移动时必须带有明显的**绿色拖尾（Trail）**效果，拖尾由粗变细并逐渐消失，模拟运动模糊。
*   **UI 界面**：
    *   画布顶部显示信息栏。
    *   左侧：“得分: [数字]”（绿色字体）。
    *   中间：矩形按钮显示“暂停”。
    *   右侧：“生命: [数字]”（红色字体）。

### 3. 交互与功能
*   **控制**：挡板水平位置跟随鼠标光标移动。
*   **游戏流程**：
    *   点击屏幕或按下空格键开始。
    *   球体撞击砖块：砖块消失，分数增加，球反弹。
    *   球体撞击墙壁/挡板：球反弹。
    *   球体掉落底部：生命值 -1，重置球位置。生命值为 0 时游戏结束。
*   **暂停功能**：点击“暂停”按钮可冻结/恢复游戏循环。

### 4. 技术实现要求
*   **文件结构**：所有代码（HTML 结构、CSS 样式、JS 逻辑）必须合并在唯一的 `index.html` 文件中。
*   **渲染引擎**：使用 HTML5 Canvas API (`ctx`) 进行 2D 绘图。
*   **动画循环**：使用 `requestAnimationFrame` 确保流畅度。
*   **无依赖**：不引入任何外部库或图片资源，所有图形通过代码绘制。
*   **自适应**：游戏画布居中显示，具有适当的边框装饰。
