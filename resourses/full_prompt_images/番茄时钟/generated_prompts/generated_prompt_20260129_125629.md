# 现代番茄时钟应用开发提示词

请作为一名前端开发专家，编写一个名为“番茄时钟”的单页Web应用。代码需合并在唯一的 `index.html` 文件中。

### 1. 整体视觉与布局
*   **风格**：极简现代风，主色调为柔和的番茄红（类似 #FF6B6B），背景为浅灰/白。
*   **布局结构**：
    *   **顶部**：应用标题“番茄时钟”及当前日期（格式：YYYY年M月D日 星期X）。
    *   **中部**：核心圆形计时器。
    *   **中下部**：三个控制按钮（开始、暂停、重置）。
    *   **底部**：三个统计卡片并排显示。

### 2. 核心组件细节
*   **圆形进度条**：
    *   使用 SVG 实现。
    *   外圈灰色轨道，内圈红色进度条。
    *   应用柔和阴影（类拟态/Neumorphism）增加立体感。
    *   圆心显示倒计时（MM:SS）及当前状态文本（如“工作中”、“已暂停”）。
*   **控制按钮**：
    *   **开始**：实心红色背景，白色文字/图标，圆角胶囊状。
    *   **暂停**：浅红色背景，深红色图标，表示挂起状态。
    *   **重置**：白色背景，红色边框，红色文字。
    *   按钮需有悬停和点击反馈效果。
*   **统计卡片**：
    *   包含三个指标：今日番茄数、专注时长（分钟）、总番茄数。
    *   卡片样式：白色背景，圆角，轻微阴影悬浮感。
    *   数字加大加粗显示。

### 3. 功能逻辑
*   **计时逻辑**：
    *   默认专注时间 25 分钟。
    *   倒计时结束自动进入 5 分钟休息（或提示切换）。
    *   SVG 进度条 `stroke-dashoffset` 随时间平滑递减。
*   **交互控制**：
    *   **开始**：启动倒计时，进度条转动。
    *   **暂停**：冻结时间与动画。
    *   **重置**：恢复初始状态（25:00）。
*   **数据持久化**：
    *   使用 `localStorage` 保存统计数据。
    *   每次完成专注周期，今日番茄+1，时长+25，总数+1。
    *   日期变更时自动重置“今日”数据。
*   **音效**：
    *   使用 HTML5 Audio API 或简单的 Oscillator 生成温和的“叮”声（开始/结束时播放）。

### 4. 技术要求
*   **文件结构**：所有 HTML, CSS, JavaScript 必须包含在一个 `index.html` 文件中。
*   **库依赖**：不使用外部库（如 Bootstrap, jQuery），仅用原生技术。
*   **响应式**：适配桌面及移动端视图。
*   **字体**：使用系统默认无衬线字体（System UI / San Francisco / Roboto）。
