# 3D Chaos Energy Visualization Prompt

### 整体效果描述
创建一个基于WebGL的3D场景，表现“混乱状态下的能量”。核心是一个发光的线框球体，向四周爆发锯齿状的电流和光束。氛围为科幻、危险（Critical Status），背景黑暗，具有强烈的辉光和故障（Glitch）艺术风格。

### 技术实现要点
*   **核心架构:** Three.js 或类似 WebGL 库。
*   **几何体:** 核心使用低多边形球体（Icosahedron），材质为发光线框（Wireframe）。
*   **着色器 (Shaders):** 这种效果依赖自定义片元着色器来生成动态的、分形的闪电纹理，而非静态贴图。
*   **后处理:** 必须包含强烈的 Bloom（泛光）效果和轻微的 Chromatic Aberration（色差/色散）。

### 交互功能说明
*   **视角控制:** 允许鼠标拖动旋转视角（OrbitControls），滚轮缩放。
*   **动态响应:** 鼠标悬停或点击核心时，电流频率加快，核心亮度增强。
*   **UI 交互:** 左上角文字面板显示实时状态，包含简单的打字机效果或闪烁警告。

### 视觉效果细节
*   **色彩:**
    *   背景: 深红至黑色的渐变虚空。
    *   核心: 亮白色高亮。
    *   能量束: 青色（Cyan）与琥珀色（Orange）交织，形成冷暖对比。
*   **元素:**
    *   **核心:** 旋转的几何线框球体。
    *   **射线:** 极坐标向外辐射的锯齿状线条，模仿闪电路径。
    *   **粒子:** 背景中漂浮的橙色正方形微粒，增加空间深度。
    *   **UI:** 红色边框文本框，内容 "Status: CRITICAL"，字体为等宽像素风。

### 动画和特效说明
*   **电流抖动:** 使用 Perlin Noise 或 Simplex Noise 驱动射线的顶点位移，产生高频颤动。
*   **脉冲:** 核心大小和亮度按心跳节奏进行无规律缩放/闪烁。
*   **故障感:** 画面偶尔出现帧错位或噪点干扰，模拟信号不稳。
*   **粒子运动:** 背景方块缓慢漂浮并随机旋转。
