# Windows 11 风格 Web OS 单文件生成提示词

你是一个世界级的前端开发专家和 UI 设计师。请编写一个包含 HTML、CSS 和 JavaScript 的单文件 (`index.html`)，创建一个功能完备、视觉精美且高度仿真的 Windows 11 风格 Web OS。

### 1. 整体视觉与布局
*   **设计语言**：严格遵循 Windows 11 设计规范。采用 Mica 材质效果（半透明磨砂玻璃）、大圆角窗口、柔和阴影和 Segoe UI（或类似无衬线）字体。
*   **壁纸**：使用 CSS 径向渐变模拟截图中蓝、橙、紫流体渐变背景，无需外部图片。
*   **桌面**：左侧网格排列应用图标。图标需使用 SVG 绘制或 FontAwesome (CDN) 模拟，包括：PyTerminal, File Explorer, VS Code, Paint, Calculator, Snake Game, Clip Editor, Settings。
*   **任务栏**：底部固定，玻璃拟态背景。中心对齐应用图标（包括开始按钮），右侧为系统托盘（时间、日期、音量、Wi-Fi）。图标悬停要有轻微上浮和背景高亮动画。

### 2. 核心交互系统
*   **窗口管理器**：
    *   实现完整的窗口生命周期：打开、关闭、最小化、最大化/还原。
    *   **拖拽与缩放**：所有窗口均可通过标题栏拖拽，通过边缘和角落调整大小。
    *   **层级控制**：点击窗口自动置顶 (z-index 管理)。
    *   **多任务**：任务栏显示当前打开的应用，点击最小化/恢复。
*   **开始菜单**：
    *   点击任务栏 Windows 图标触发，中央弹出。
    *   包含顶部搜索栏、中间“已固定”应用网格、底部用户头像和电源按钮。
    *   添加平滑的淡入淡出和缩放动画。

### 3. 功能模块与应用实现
请在代码中实现以下具体应用，每个应用在独立窗口中运行：

*   **File Explorer (文件资源管理器)**：
    *   UI：左侧导航栏（Desktop, Pictures, Videos等），顶部面包屑导航，右侧内容区域显示文件网格。
    *   功能：模拟文件系统，可点击文件夹进入子目录，双击文件打开对应应用（如双击 .txt 打开记事本）。
*   **PyTerminal (模拟终端)**：
    *   UI：黑色背景，绿色/白色等宽字体，类似截图中的命令提示符风格。
    *   功能：模拟 Python shell。支持输入命令（如 `print()`, `help`, `cls`, `dir`），并给予虚假的执行反馈。
*   **VS Code (代码编辑器)**：
    *   UI：深色主题，左侧活动栏，中间代码编辑区。
    *   功能：基于 `textarea` 或 `contenteditable` 的文本编辑，提供基础的语法高亮模拟（HTML/JS）。
*   **Paint (画图)**：
    *   使用 HTML5 `<canvas>` 实现。支持画笔颜色选择、笔刷大小调整、橡皮擦功能。
*   **Snake Game (贪吃蛇)**：
    *   在窗口内实现完整的贪吃蛇游戏逻辑，包含计分板和键盘控制。
*   **Calculator (计算器)**：
    *   实现标准计算器功能的 UI 和逻辑。
*   **Settings (设置)**：
    *   展示模拟的系统信息或允许更改壁纸颜色的简单选项。

### 4. 技术实现细节
*   **单文件架构**：CSS 必须写在 `<style>` 标签内，JavaScript 必须写在 `<script>` 标签内，不引用任何本地文件。可以使用可靠的 CDN（如 FontAwesome, Google Fonts）。
*   **代码质量**：使用现代 ES6+ JavaScript，逻辑清晰，模块化编写（例如将每个 App 封装为类或对象）。
*   **动画效果**：大量使用 CSS transitions 和 keyframes 实现窗口打开时的缩放效果、按钮点击波纹效果、菜单滑出效果，确保 60fps 流畅度。
*   **响应式**：确保在 Chrome 浏览器全屏显示时布局完美，禁止页面整体滚动。

请生成这段代码，确保我复制粘贴后直接能在浏览器中体验一个“超出预期”的 Web OS。
